# Student API + Next.js Frontend

This project is a full-stack Student management app built with Next.js and Tailwind CSS.

## Local Setup
1. Install dependencies:
```bash
npm install
```

2. Configure environment:
- Copy `.env.example` to `.env.local`
- Fill in MongoDB values

3. Run:
```bash
npm run dev
```

## API Endpoints
- `GET /api/students`
- `POST /api/students`
- `GET /api/students/:id`
- `PUT /api/students/:id`
- `DELETE /api/students/:id`

## Frontend Features
- Student list table
- Add/Update form
- Delete with confirmation
- Tailwind CSS styling
